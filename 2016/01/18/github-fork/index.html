Fork 通俗简介

2016-01-18 |

发现仓库

你在站台等高铁，闲的无聊于是就打开笔记本在网上逛逛，发现有一个妹子在 GitHub 上建了一个名为 FuLi 的仓库，仓库里存了一些视屏、音乐和诗词歌赋等等。

Fork 仓库

你觉得这个仓库太有趣了，于是以迅雷不及掩耳盗铃之势点了 GitHub 上的 fork 按钮，轻松复制妹子的 FuLi 仓库到自己的 GitHub 帐号下。

Clone 仓库

一看表快发车了，悲催的是这趟车上没网，但你还想继续看这个仓库，于是你 git clone 命令一敲，就把自己 GitHub 账号下的 FuLi 仓库内容存储到笔记本上了，旅途不再寂寞。

更新文件

你在欣赏诗词歌赋的时候发现了一些错别字，作为一个有情怀的强迫症患者当然要把这些错别字统统改掉。

提交更改

刚改完错别字，还没来得及欣赏视频和音乐，高铁到站了。先就这样吧，git commit 一敲，让 Git 记录我的改动。

上传更改

到了酒店，又有网了，你迫不及待地将你的更改 push 到自己 GitHub 上的 FuLi 仓库里。

通知更改

做了好事当然要得瑟一下，于是你给在 GitHub 上给妹子发送了一个 pull request ，跟妹子说哥哥帮你修改了错别字。

整合更改

妹子一看改得不错，接受了你的 pull request，并把你的修改整合到她的 FuLi 仓库里，还不忘感谢你是一个好人。

取出变化文件

很多人都对妹子的 FuLi 仓库感兴趣，其中有异国友人还为视频做了字幕，也被妹子整合进她的 FuLi 仓库了。考虑到茴字有四种写法，你对异国友人的字幕不太放心，决定 git pull 到笔记本上看看。

本地与云端同步

原来那位异国友人是个 ABC，家学渊源，字幕做得不错。你一边默念好人一生平安，一边敲下 git push 把字幕同步到自己 GitHub 上的 FuLi 仓库里。
