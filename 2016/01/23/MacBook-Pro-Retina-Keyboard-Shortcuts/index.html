显示桌面

Command–Mission Control（Mission Control 就是标 F3 那个键）

区域截屏

Shift-Command-4 Save picture of selected area as a file
Shift-Command-4-Space Take a screen shot of a window as a file
Control-Shift-Command-4 Save picture of selected area to the clipboard
Control-Shift-Command-4-Space Take a screen shot of a window to the clipboard

截屏

Shift-Command-3 Save picture of screen as a file
Control-Shift-Command-3 Save picture of screen to the clipboard
